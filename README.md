# A-Z Handwritten Alphabet Recognition

## Project Overview
This project implements and compares three deep learning models for handwritten alphabet recognition using the A–Z Handwritten Alphabet Dataset.

The objective is to classify handwritten English letters (A–Z) by evaluating different neural network architectures and comparing their performance.

## Models Implemented

- TensorFlow Artificial Neural Network (ANN)
- Keras Artificial Neural Network (ANN)
- Keras Convolutional Neural Network (CNN)

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Dataset

A–Z Handwritten Alphabet Dataset

## Results

| Model | Test Accuracy |
|--------|--------------:|
| TensorFlow ANN | 98.06% |
| Keras ANN | 98.58% |
| Keras CNN | **98.95%** |
